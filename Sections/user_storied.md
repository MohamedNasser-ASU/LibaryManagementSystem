# 2. User Stories

## 2.1 User Story 1
**Card:** As a student, I want to search for textbooks by subject, so that I can easily find the exact resources I need for my classes.

**Acceptance Criteria:**
* The system should display a search bar on the home screen.
* The system should display the result of the search whether the book is available or not.

## 2.2 User Story 2
**Card:** As a librarian, I want to organize the textbooks based on field of study, so that resources are well organized and cataloged.

**Acceptance Criteria:**
* The system should have ”tagging” as a function to allow librarians put books under certain categories.
* The system must provide a filtering option so users can browse and view all resources grouped under the same category.

## 2.3 User Story 3
**Card:** As an administrator, I want to set a fee assessed for damaged books, so that borrowers take better care of resources.

**Acceptance Criteria:**
* The system should have an online payment system so users can pay cashless with their bank accounts.
* The system must provide a settings page where the penalty fee amount can be inputted and updated.
