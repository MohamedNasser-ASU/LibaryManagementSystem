## System Description
This library management system (LMS) will make it easier for the university students to borrow books by automating the borrowing, returning and cataloging actions. It’s designed for use by students, librarians and administrators, providing a secure, simple, easy to use and efficient way for all users to benefit from it and manage the library resources.
